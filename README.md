# SecurePass Generator

A modern, client-side web application for generating strong, secure, and customizable passwords. Built with simplicity and security in mind, this tool helps you create robust passwords to enhance your online security.

**Live Demo:** [https://rajaryan77-art.github.io/SecurePass-Generator/](https://rajaryan77-art.github.io/SecurePass-Generator/)

*   **Customizable Length:** Generate passwords from **X to Y characters** long.
*   **Character Set Control:** Fine-tune your password by including or excluding:
    *   Uppercase letters (A-Z)
    *   Lowercase letters (a-z)
    *   Numbers (0-9)
    *   Special Symbols (e.g., ! @ # $ %)
*   **One-Click Generation:** Instantly create a new password with your settings.
*   **One-Click Copy:** Easily copy the generated password to your clipboard for immediate use.
*   **Strength Indicator:** A visual indicator shows the estimated strength of your generated password[citation:5].
*   **Client-Side Security:** Everything runs in your browser. **No passwords are ever sent to any server**, ensuring complete privacy[citation:6].

## üöÄ How to Use

1.  Open the [Live Demo](https://rajaryan77-art.github.io/SecurePass-Generator/).
2.  Use the slider or input field to select your desired password length.
3.  Check the boxes for the character types you want to include in your password (e.g., Symbols, Numbers).

5.  Your new secure password will appear in the display box. Click the **"Copy"** button next to it to copy it to your clipboard.

**Tip:** For a strong password, use a length of at least 12 characters and include all four character types[citation:4].
